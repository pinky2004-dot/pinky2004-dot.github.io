Projects

Pet Health AI

Pet Health AI is an AI-powered web application that helps pet owners assess whether their pet's condition is an emergency or not. It uses Amazon Bedrock to classify user messages based on urgency, and Amazon SageMaker to analyze uploaded images of skin conditions (starting with dogs). If an emergency is detected, the app redirects users to nearby veterinary services using AWS Location Service. Built with Flask for the backend, React for the frontend, and integrated with AWS Cognito for secure user authentication, it also leverages Pinecone for semantic search within a RAG pipeline.
